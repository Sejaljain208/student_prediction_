Instructions

Upload a CSV file containing student data. The file must include the following columns:

  • student_id - Unique student identifier
  • name - Student full name
  • grade - Grade level (9-12)
  • section - Section/Class (A, B, C, etc.)
  • email - Student email
  • phone - Contact phone number
  • age - Student age
  • gender - Gender (M/F)
  • attendance_rate - Attendance rate (0.0 - 1.0)
  • gpa - GPA (0.0 - 4.0)
  • parent_education - Parent education level
  • family_income - Family income level (Low/Medium/High)
  • study_hours_per_week - Weekly study hours
  • extracurricular_activities - Number of activities
  • previous_failures - Number of previous failures
